# Ritual — Ecommerce de Beauty & Self Care

Ecommerce realizado con PHP + MySQL para Programación II

## Tecnologías

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript (Vue 3 desde CDN)
- Bootstrap 5 (solo grid)

## Funcionalidades

- Registro de usuarios con dos roles: Cliente y Vendedor
- Login con manejo de sesiones
- Logout
- Catálogo de productos
- Detalle de producto con información del vendedor asociado
- Gestión de productos (creación)
- Sistema de pedidos
- Listado de pedidos por usuario

### Pasos

1. Copiar la carpeta `ritual` dentro de `C:\xampp\htdocs\`
2. Iniciar Apache y MySQL desde XAMPP
3. Abrir phpMyAdmin: `http://localhost/phpmyadmin/`
4. Crear una nueva base de datos llamada `ritual`
5. Seleccionar la base `ritual` y entrar a la pestaña **Importar**
6. Importar el archivo `ritual.sql`
7. Abrir el navegador en: `http://localhost/ritual/usuarios/login.html`
